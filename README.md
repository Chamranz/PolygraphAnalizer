## Описание проекта

Это программа для анализа данных полиграфа, которые уже подверглись первичной обработке и представлены в виде метрик, описывающих психофизиологические реакции через эмоциональные характеристики. Результаты выводятся в формате Excel.

Каждая эмоция описывается двумя параметрами: вовлечённость и эмоциональный знак .

Вовлечённость — это, грубо говоря, сила переживаемой эмоции.
Эмоциональный знак указывает на её модальность: положительная или отрицательная была реакция.
Важно понимать, что ключевое значение имеет не абсолютное значение этих показателей, а их динамика во времени . Поэтому анализ проводится по временным интервалам, изучается характер изменений. Например, если вовлечённость возрастает, а эмоциональный знак снижается, это может свидетельствовать о сильных негативных эмоциях, таких как гнев или раздражение. Аналогичным образом можно интерпретировать и другие комбинации изменений.

Для анализа динамики обычно используется линия тренда — аппроксимация полиномом первой степени. Однако такой подход эффективен только при анализе коротких эпизодов (десятки секунд). При работе с более длительными отрезками, например, при анализе фильма, требуется полином более высокой степени.

Построение модели связано с определёнными трудностями, связанными с выбором оптимальной степени полинома. В качестве решения я предложил формулу для расчёта степени полинома:
ln(duration / 10) .

Это монотонно возрастающая функция с постепенным затуханием роста, что позволяет степени полинома увеличиваться, но не слишком быстро. Деление на 10 обусловлено скоростью протекания психофизиологических процессов, регистрируемых полиграфом.

Имеются функции как для анализа по нескольким заданиям/эпизодов для одного респондента, так идля сравнения разных респондентов по ряду задании/эпизодов. Также программа добавляет анимированные графики с 
динамически обновляющейся линией тренда за последние 5 секунд. Все примеры есть в ветке main.
